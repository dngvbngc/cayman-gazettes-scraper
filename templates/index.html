Scrape Cayman Gazettes

To retrieve the token:
1. Visit the official Cayman Gazettes site
2. Right click, select 'Inspect'
3. Select the 'Network' tab at the top of the page
4. Reload the Gazettes site by pressing Command + Shift + R
5. On the "Name" left bar, select any Network call beginning with 'items'
6. Click on Payload on the right pannel
7. Copy channel token
